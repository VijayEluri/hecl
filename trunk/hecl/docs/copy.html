copy
 Hecl Commands 

Name

copy — Return a copy of the variable.

Synopsis

copy varname

Description

The copy command is the equivalent of dollar substitution. It returns a copy of the variable in question, meaning that other commands that operate on the copy won't modify the original value.

Example

set foo "bee"
append [copy foo] "bop"
puts $foo

Produces:

bee
