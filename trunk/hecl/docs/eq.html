eq
 Hecl Commands 

Name

eq — Tests string equivalence.

Synopsis

eq string1 string2

Description

The eq commands compares two strings, returning 1 if they are equal, 0 if they are not.

Example

if {eq 1 1.0} {
    puts "True"
} else {
    puts "False"
}

Produces:

False

Despite being numerically equivalent, the strings "1" and "1.0" are different.
