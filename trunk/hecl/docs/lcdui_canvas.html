Name

lcdui.canvas — Creates a canvas

Synopsis

lcdui.canvas [-title title] [-suppresskeys 1 | 0 ] [-fullscreen 1 | 0 ] [-autoflush 1 | 0 ] [-cmdbg 1 | 0 ]

Description

The lcdui.canvas command creates a canvas that can be used to draw arbitrary items, such as rectangles, text, circles, and respond to keyboard events.
