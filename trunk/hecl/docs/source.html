source
 File Interaction 

Name

source — Evaluate Hecl script in a file.

Synopsis

source filename

Description

The source command evaluates the Hecl script located in file filename.

Example

# Variable foo is defined as "Hello world" in foo.hcl
source foo.hcl
puts $foo

Produces:

Hello world
